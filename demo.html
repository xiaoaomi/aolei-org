02 · Kiro 全生命周期
「文字一敲，云动了，模型换了，龙虾🦞活了。」

从部署上线到模型切换，从系统运维到玄学任务，Kiro 把自然语言变成云上操作。 只需一句话，一切就绪。

SCENE_01
🚀 部署上线
告诉 Kiro「帮我在 AWS 上部署 OpenClaw」——选 Region、机型、模型，Kiro 自动跑，VPC / EC2 / IAM / VPC Endpoints 一键创建，约 8 分钟 OpenClaw 上线，全程不用打一行命令。
「云的弹性，因语言而触手可及。」
SCENE_02
🔄 切换模型
「帮我把模型换成 DeepSeek」——Kiro 秒改 openclaw.json 配置、重启 Gateway、验证响应，Bedrock 上 Nova / Claude / Kimi / DeepSeek 随时切换，零停机。
「换模型，像换频道一样简单。」
SCENE_03
🦞 复活龙虾
「我的龙虾不回应了，帮我排查一下」——Kiro SSH 进服务器，读日志、定位 config 非法字段、一键修复配置、重启 Gateway，30 秒内复活。
「没有 Kiro 驯服不了的故障。」
